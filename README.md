# AI Content System
Платформа для генерации контента с помощью искусственного интеллекта.

##  О проекте

**AI Content System** — это веб-приложение, которое позволяет автоматически создавать различные виды контента с использованием AI:

* ✍️ тексты
* 📚 задания
* ❓ вопросы
* 📧 письма
* 🧠 планы
* ➗ математические решения
* 🖼 анализ изображений

## 🛠 Технологии

* **Backend:** Spring Boot (Java)
* **Frontend:** HTML, CSS, JavaScript
* **База данных:** H2 / (или твоя)
* **AI API:** OpenAI / Groq / Gemini

## 🔐 Безопасность

* API ключи не хранятся в репозитории
* Используется `.gitignore` для исключения конфиденциальных данных

## ⭐ Заключение

Проект демонстрирует создание полноценной AI-системы генерации контента с использованием современных технологий и архитектуры.
